### Guidelines

- Never stretch or distort the logo mark
- Maintain a clear-space equal to the logo height on all sides
- Do not recolour the mark — only `teal-500` on white/dark backgrounds
- Minimum size: 32 × 32 px

## Typography

**Primary font:** [Inter](https://rsms.me/inter/) — Open Source, system-level rendering quality.

<Typeset
  fontSizes={[12, 14, 16, 18, 20, 24, 30, 36, 48, 60, 72]}

  fontFamily="Inter, ui-sans-serif, system-ui, sans-serif"

### Weight Scale

| Weight | Class | Usage |
|--------|-------|-------|
| 300 | `font-light` | Captions, secondary labels |
| 400 | `font-normal` | Body text, descriptions |
| 500 | `font-medium` | Navigation, emphasis |
| 700 | `font-bold` | Headings, CTAs, numeric values |

### Type Scale

| Name | Size | Line Height | Usage |
|------|------|-------------|-------|
| `text-xs` | 12px | 16px | Labels, badges, table cells |
| `text-sm` | 14px | 20px | Body secondary, helper text |
| `text-base` | 16px | 24px | Body primary |
| `text-lg` | 18px | 28px | Sub-headings |
| `text-xl` | 20px | 28px | Section headings |
| `text-2xl` | 24px | 32px | Page sub-titles |
| `text-4xl` | 36px | 40px | Section titles |
| `text-5xl` | 48px | 1 | Hero headings |

## Voice & Tone

| Principle | Do | Don't |
|-----------|-----|-------|
| **Precise** | "Win rate: 68.2%" | "Doing pretty well" |
| **Direct** | "Deploy strategy" | "Click here to get started with deploying" |
| **Confident** | "Your bots are live" | "Your bots might be running" |
| **Human** | "Something went wrong. Try again." | "Error 500: Internal Server Error" |

### Numeric formatting

- PnL: always include sign — `+$1,240` / `−$320`
- Percentages: one decimal place — `12.4%`
- Large numbers: thousands separator — `$1,240,000`
- Crypto quantities: up to 6 decimal places — `0.004821 BTC`
